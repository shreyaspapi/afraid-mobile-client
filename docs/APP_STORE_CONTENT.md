## Privacy Policy (Required)

You'll need to create a `PRIVACY_POLICY.md` file. Here's a template:

```markdown
# Privacy Policy for Parity

**Last Updated:** [DATE]

## Data Collection

Parity does not collect, transmit, or share any personal information or usage data. All data remains on your device.

## Data Storage

- Server credentials (IP addresses and API keys) are stored encrypted on your device
- All cached server data is stored locally
- No data is transmitted to third parties

## Network Communication

Parity communicates directly with your Unraid server(s) using the IP addresses and API keys you provide. No intermediary services are used.

## Third-Party Services

Parity does not use any third-party analytics, advertising, or tracking services.

## Your Rights

You can delete all app data at any time by uninstalling the app or clearing app data in your device settings.

## Changes

We may update this privacy policy. Changes will be posted in the app and on our GitHub repository.

## Contact

For questions about privacy, please contact: [YOUR_EMAIL] or open an issue on GitHub.
```

## Tips for Approval

### Apple App Store
1. **Provide detailed review notes** explaining how to test
2. **Include demo mode** so reviewers can test without a server
3. **Ensure privacy policy** is accurate and accessible
4. **Follow Human Interface Guidelines**
5. **Test thoroughly** on all supported devices
6. **Respond quickly** to reviewer questions

### Google Play
1. **High-quality screenshots** with text overlays
2. **Feature graphic** (1024x500px) required
3. **Complete the content rating questionnaire** honestly
4. **Target latest Android SDK** when possible
5. **Provide testing instructions** in developer notes
